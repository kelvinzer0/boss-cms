This is an example of a markdown file that demonstrates various markdown elements. You can use this file to test your React `Content` component.

## Headings

Markdown supports different levels of headings:

### Subheading 1
#### Subheading 2
##### Subheading 3

## Paragraphs

Here is a simple paragraph. Markdown makes it easy to write clean and readable text. You can add **bold**, _italic_, or even ~~strikethrough~~ text.

Another paragraph with a link to [Google](https://www.google.com).

> > Nested blockquotes are also supported.

## Code Blocks

Inline code can be written using backticks like this: `console.log('Hello, World!')`.

For multi-line code blocks, use triple backticks:

```javascript
function greet(name) {
  return `Hello, ${name}!`;
}

console.log(greet("World"));
